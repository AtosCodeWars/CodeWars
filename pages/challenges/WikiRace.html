Wiki Race page

Instructions: Welcome to the Wiki Race Challenge. The objective of the game is to traverse from one wikipedia page to another wikipedia page. The catch is that you can only move through page using the Wikipedia's Hyperlinks text. You may not use the search bar or edit the wikipedia article. Once you found a link between the initial page and final page you need to write it down in the team's breakout chat. See the example below:

You will need to complete the following three problems to finish the challenge. Good Luck!
